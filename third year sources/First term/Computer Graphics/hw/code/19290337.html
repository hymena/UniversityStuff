You can both use buttons and keyboard to interact with letters. The key 'w' on keyboard moves the letters up, 's' moves the letters down, 'a' moves the letters left and 'd' moves the letters right. Also the key '+' on keyboard increases the size of letters and '-' decreases the size of letters. Also button names are given descriptive so you can easily understand and use them.
